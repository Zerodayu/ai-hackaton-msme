# Project for Ai Hackaton

### Requirements
- [Bun](https://bun.com/)

### Clone Project
```
git clone https://github.com/Zerodayu/ai-hackaton-msme.git
```

### Install Dependencies
```
cd ai-hackaton-msme
```
```
bun i
```

### Setup Environments
> find the `env.sample` file the create `.env` file on root directory and paste your database url(NeonDb)

### Run Dev Server
```
bun dev
```
